Type the law.

**Train your fingers. Master the law.**

Verba Law is a professional blind typing trainer designed specifically for legal practice.  
It helps develop accuracy, speed, and muscle memory using real legal language.

Features:
- Structured lessons based on legal vocabulary  
- Real legal texts for advanced practice  
- Error tracking with "hard words" training  
- Free typing mode and custom text input  
- Visual keyboard with finger guidance  
- Clean, distraction-free interface  

Why this project?
Typing is not just about speed — it's about thinking.

For legal professionals:
- faster typing = faster drafting  
- fewer errors = better documents  
- no keyboard focus = better concentration  

This trainer is built to support real legal work.

How it works
- Start with short words and basic legal terms  
- Move to structured legal phrases and norms  
- Practice full legal texts  
- Train difficult words separately  
- Build automatic typing навыки (muscle memory)

Tech Stack
- HTML  
- React (via CDN)  
- Tailwind CSS  
- No backend (frontend-only MVP)

Installation
No installation required.
